# AX 실전 프로젝트 과정 — 신청 페이지

풀무원 전사 AI 공개교육 「AX 실전 프로젝트 과정」 신청 랜딩페이지입니다.
단일 HTML 파일로 제작되어 별도 빌드 없이 바로 열어보고 배포할 수 있습니다.
신청은 **Microsoft Forms**로 받고, 응답은 **OneDrive 엑셀에 자동 저장**됩니다.

## 파일 구성

| 파일 | 설명 |
|------|------|
| `index.html` | 신청 페이지 본체 (히어로·과정소개·커리큘럼·강사·신청폼·FAQ·푸터) |
| `README.md` | 본 문서 (연동·배포·수정 가이드) |

## 바로 미리보기

`index.html` 파일을 더블클릭하면 브라우저에서 바로 열립니다.

## 신청 폼(Microsoft Forms) 연동 구조

- 신청 섹션에는 Microsoft Forms가 **iframe으로 삽입**되어 있습니다.
  - 현재 연결된 폼: `https://forms.office.com/r/8dmY5awAJ9`
- 신청자가 폼을 제출하면, 응답은 MS Forms와 연결된 **OneDrive 엑셀 파일에 실시간으로 추가**됩니다.
- 폼 응답 엑셀은 Microsoft Forms의 **「응답」 탭 → 「Excel에서 열기」** 로 만들어진 OneDrive 파일에서 확인합니다.

### 폼을 다른 것으로 교체하려면
`index.html` 에서 `forms.office.com/r/8dmY5awAJ9` 를 검색해 **두 군데**(iframe `src`, 하단 새 창 링크)의 폼 ID를 새 폼 주소로 바꾸면 됩니다.

### 임베드가 빈 화면으로 보일 때 (중요)
- 폼 설정이 **「조직 내 사용자만 응답」** 인 경우, 보는 사람이 풀무원 M365 계정에 **로그인되어 있어야** 임베드 폼이 표시됩니다.
  - 사내 임직원은 보통 이미 로그인되어 있어 바로 보입니다.
  - 로그인되어 있지 않으면 임베드가 비어 보일 수 있는데, 이를 대비해 폼 아래에 **「새 창에서 신청서 열기」** 링크를 넣어 두었습니다(새 창에서는 로그인 후 정상 작성 가능).
- 로그인 없이 **누구에게나** 폼을 바로 보여주려면, MS Forms 설정에서 응답 대상을 **「누구나 응답 가능」** 으로 변경하세요. (단, 외부 응답도 허용됨)

## 배포 (외부 공개)

`index.html` 파일 하나만 올리면 됩니다.

- **사내 공유**: 사내 포털/그룹웨어 게시판이나 공유 드라이브에 올리고 링크 공유.
- **Netlify**: [app.netlify.com/drop](https://app.netlify.com/drop) 의 Drop 영역에 `course-signup` 폴더를 끌어다 놓으면 즉시 URL 발급.
- **Vercel**: [vercel.com](https://vercel.com) → 새 프로젝트 → 폴더 업로드.
- **GitHub Pages**: 레포에 올리고 Settings → Pages 에서 활성화.

`index.html` 에서 아래 내용을 검색해 직접 수정할 수 있습니다.

| 문의처 / 담당자 | 푸터 (`yejeon.choi@pulmuone.com`) |

## 사용 기술

- 단일 HTML + Tailwind CSS (Play CDN) + Pretendard 웹폰트
- 바닐라 JavaScript (헤더 효과·스크롤 애니메이션·숫자 카운트업)
- Microsoft Forms 임베드 — 신청 접수 및 OneDrive 엑셀 저장

> Tailwind Play CDN은 콘솔에 "프로덕션용 아님" 경고를 표시합니다.
> 사내 안내 페이지 용도로는 문제없으며, 대규모 운영 시 빌드형(Tailwind CLI)으로 전환할 수 있습니다.
